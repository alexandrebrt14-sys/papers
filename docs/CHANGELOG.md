## [2.0.0] — 2026-03-24

### Adicionado
- Sistema FinOps completo com tracking de custos por token/plataforma
- Dashboard HTML centralizado com painel por plataforma (OpenAI, Anthropic, Gemini, Perplexity)
- Governança de gastos: limites mensais/diários, hard stop em 95%, alertas automáticos
- Alertas por email via Resend API (70% warning, 90% critical, 100% exceeded)
- Documentação automatizada: REQUIREMENTS.md, CHANGELOG.md, GOVERNANCE.md, ARCHITECTURE.md
- Script update-docs.py para atualização automática pós-coleta
- GitHub Action para atualização de documentação
- Especificação de requisitos formalizada (RF-01 a RF-09, RNF-01 a RNF-05)

### Modificado
- geo_daily.py integrado com FinOps check
- citation_monitor.py agora registra custos e verifica budgets antes de cada query
- geo_db.py expandido com 3 tabelas FinOps (finops_usage, finops_budgets, finops_alerts)

## [1.0.0] — 2026-03-24

### Adicionado
- Infraestrutura inicial com 7 módulos de coleta
- Citation Tracker (Module 1): monitoramento em 5 LLMs com 17 campos
- Competitor Benchmark (Module 2): grupo de controle com entidades comparáveis
- SERP vs AI Overlap (Module 3): divergência Google SERP vs fontes IA
- Content Intervention Tracker (Module 4): A/B testing de conteúdo
- Time Series Persistence (Module 5): snapshots diários para análise longitudinal
- Statistical Analysis (Module 6): chi-squared, t-test, ANOVA, regressão, Bonferroni
- Citation Context Analyzer (Module 7): sentimento, atribuição, precisão factual
- Schema SQL com 8 tabelas e 14 índices
- CLI com 9 comandos (click + rich)
- 19 testes unitários (pytest)
- GitHub Actions: daily-collect (06:00 BRT) + weekly-benchmark (domingo 08:00)
- Configuração para 5 LLMs e 55 queries padronizadas
- Visualizações publication-quality (5 tipos, 300 DPI)
